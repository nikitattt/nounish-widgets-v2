# Nouns Widgets V1

![header](https://nounishimg.iamng.wtf/widgets-retro-and-infra-support/header.jpg)

## Structure

```
/
├─ apps/
│   ├─ backend   → API that loads & transforms auction, image, and governance data
│   └─ website   → Installation pages + prod widget code
└─ widgets/      → Scriptable-compatible JavaScript for each widget(non-prod)
```
